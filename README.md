# 🥁 Барабаны сепараторов

Веб-приложение для учёта и замены барабанов на сепараторах. Данные синхронизируются через облачную базу данных (Supabase / Clever Cloud PostgreSQL).

## 🚀 Возможности

- 📋 **Учёт сепараторов** — просмотр списка сепараторов с установленными барабанами (верх/низ).
- 🔄 **Замена барабанов** — быстрая замена барабана через всплывающее окно.
- 📦 **Управление складом** — добавление и удаление барабанов из общего списка.
- 📜 **История замен** — автоматическое сохранение всех действий с датой и временем.
- ☁️ **Синхронизация** — данные хранятся в облачной базе и доступны с любого устройства.
- 📊 **Экспорт в Excel** — выгрузка актуальных данных в файл `.xlsx`.
- 📱 **Адаптивный дизайн** — удобная работа на компьютерах, планшетах и смартфонах.

## 🛠️ Технологии

- **Frontend:** HTML, CSS, JavaScript (чистый, без фреймворков)
- **База данных:** PostgreSQL (Supabase / Clever Cloud)
- **Библиотеки:**
  - [SheetJS](https://sheetjs.com/) — для работы с Excel-файлами
  - [Supabase JS](https://supabase.com/docs/reference/javascript) — для подключения к базе данных

## 🗄️ Структура базы данных

### Таблица `separators`
| Колонка | Тип | Описание |
|---------|-----|----------|
| `id` | SERIAL | Уникальный идентификатор |
| `num` | TEXT | Номер сепаратора |
| `top` | TEXT | Барабан сверху |
| `bottom` | TEXT | Барабан снизу |
| `created_at` | TIMESTAMP | Дата создания записи |

### Таблица `drums`
| Колонка | Тип | Описание |
|---------|-----|----------|
| `id` | SERIAL | Уникальный идентификатор |
| `name` | TEXT | Название барабана |
| `created_at` | TIMESTAMP | Дата создания записи |

### Таблица `history`
| Колонка | Тип | Описание |
|---------|-----|----------|
| `id` | SERIAL | Уникальный идентификатор |
| `date` | TEXT | Дата и время события |
| `text` | TEXT | Текст события (замена) |
| `created_at` | TIMESTAMP | Дата создания записи |
